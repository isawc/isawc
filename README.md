[![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)](https://github.com/isawc/cv-insight-api)
[![FastAPI](https://img.shields.io/badge/FastAPI-009688?style=for-the-badge&logo=fastapi&logoColor=white)](https://github.com/isawc/cv-insight-api)

- 🌐 Deploy em produção no Railway
- 📚 Documentação automática com Swagger
